# WPCG 1.0 alpha
============

WordPress Content Generator

1. Always back up your database first
2. Enter (DB) connection parameters
2. Select the tab (depending on the desired data type)
3. Fill in all required fields
4. *Set parameters for generating (enumeration from-to)*
4. Generate data!

**Warning: use at your own risk ;-) Tested on WordPress 4.2.4...**
